Kilo
===

Kilo is a small text editor in less than 1K lines of code (counted with cloc).


Usage: kilo `<filename>`

Keys:

    CTRL-S: Save
    CTRL-Q: Quit
    CTRL-F: Find string in file (ESC to exit search, arrows to navigate)
